2. The rmg.constants Module¶

Contains a number of constants to be used throughout RMG. These can be divided into physical constants and global parameters. Note that RMG uses SI units throughout, so all physical values are stored in combinations of m, s, kg, mol, etc. Accordingly, all energies are stored in J.

rmg.constants.Na¶

The Avogadro constant

rmg.constants.Na = 6.02214179e+23

rmg.constants.R¶

The molar gas constant, in J/mol/s

rmg.constants.R = 8.314472

rmg.constants.c¶

The speed of light in a vacuum

rmg.constants.c = 299792458.0

rmg.constants.h¶

The Planck constant

rmg.constants.h = 6.62606896e-34

rmg.constants.kB¶

The Boltzmann constant

rmg.constants.kB = 1.3806504e-23
