10. The rmg.species Module¶

Contains classes describing chemical species.

10.1. Classes in rmg.species¶

10.1.1. Species¶

class rmg.species.Species(id=0, label='', structure=None, reactive=True, SMILES=None)¶

Represent a chemical species (including all of its resonance forms). The attributes are:

Attributes Description
id A unique integer identifier
label A more descriptive (but not necessarily unique) string label
lennardJones The Lennard-Jones parameters for the species
reactive True if the species is reactive, False if inert
spectralData The spectral data (degrees of freedom) for the species
structure A list of structure.Structure objects representing the set of resonance isomers
thermoData The thermodynamic parameters for the species, always a derived class of thermo.ThermoData
thermoSnapshot The thermodynamic parameters at the current point in the simulation
findSubgraphIsomorphisms(other)¶
Returns a list of the subgraph matches between the species and the functional group other.
fromAdjacencyList(adjstr)¶
Convert an adjacency list string adjstr to a Species object.
fromCML(cmlstr)¶
Convert a string of CML cmlstr to a Species object.
fromInChI(inchistr)¶
Convert an InChI string inchistr to a Species object.
fromOBMol(obmol)¶
Convert an OpenBabel OBMol object obmol to a Species object.
fromSMILES(smilesstr)¶
Convert a SMILES string smilesstr to a Species object.
generateThermoData()¶

Generate thermodynamic data for the species using the thermo database.

Generates the thermo data for each structure (resonance isomer), picks that with lowest H298 value, and saves it to self.thermoData.

getEnthalpy(T)¶
Return the enthalpy of the species at temperature T.
getEntropy(T)¶
Return the entropy of the species at temperature T.
getFormula()¶
Return the chemical formula for the species.
getFreeEnergy(T)¶
Return the Gibbs free energy of the species at temperature T.
getHeatCapacity(T)¶
Return the heat capacity of the species at temperature T.
getResonanceIsomers()¶
Generate all of the resonance isomers of this species. The isomers are stored as a list in the structure attribute. If the length of structure is already greater than one, it is assumed that all of the resonance isomers have already been generated.
getThermoData()¶

Get thermodynamic data for the species, generating it if necessary.

The specific type of thermo data returned is unknown, but it will be a subclass of thermo.ThermoData. If self.thermoData does not yet exist, it is created using generateThermoData().

isIsomorphic(other)¶
Returns True if the two species are isomorphic and data:False otherwise.
isSubgraphIsomorphic(other)¶
Returns True if the species is isomorphic with the other functional group and data:False otherwise.
toAdjacencyList()¶
Convert a Species object to an adjacency list.
toCML()¶
Convert a Species object to CML.
toCantera()¶
Return a Cantera ctml_writer instance
toInChI()¶
Convert a Species object to an InChI string.
toOBMol()¶
Convert a Species object to an OBMol object.
toSMILES()¶
Convert a Species object to an InChI string.

10.1.2. ThermoDatabase¶

class rmg.species.ThermoDatabase¶

Represent an RMG thermodynamics database.

contains(structure)¶
Search the dictionary for the specified structure. If found, the label corresponding to the structure is returned. If not found, None is returned.
getThermoData(structure, atom)¶
Determine the group additivity thermodynamic data for the atom atom in the structure structure.
load(dictstr, treestr, libstr)¶
Load a thermodynamics group additivity database. The database is stored in three files: dictstr is the path to the dictionary, treestr to the tree, and libstr to the library. The tree is optional, and should be set to ‘’ if not desired.
toXML()¶
Return an XML representation of the thermo database.

10.1.3. ThermoDatabaseSet¶

class rmg.species.ThermoDatabaseSet¶

A set of thermodynamics group additivity databases, consisting of a primary database of functional groups and a number of secondary databases to provide corrections for 1,5-interactions, gauche interactions, radicals, rings, and other functionality. There is also a primary library containing data for individual species.

getThermoData(struct)¶
Determine the group additivity thermodynamic data for the structure structure.
load(datapath)¶
Load a set of thermodynamics group additivity databases from the general database specified at datapath.
saveXML(datapath)¶
Save the loaded databases in the set to XML files.

10.1.4. ThermoSnapshot¶

class rmg.species.ThermoSnapshot(temperature=0.0, heatCapacity=0.0, enthalpy=0.0, entropy=0.0)¶

A set of thermodynamic state variables for a given state under a single set of conditions. During dynamic simulations, this class enables the simulator to update the thermodynamics of each species one per iteration. The attributes are:

Attribute Description
temperature the temperature at which the snapshot was taken in K
heatCapacity the heat capacity at the current conditions in J/mol*K
enthalpy the enthalpy at the current conditions in J/mol
entropy the entropy at the current conditions in J/mol*K
freeEnergy the Gibbs free energy at the current conditions in J/mol
isValid(temperature)¶
Return True if the current snapshot is still valid at the new temperature (in K), or False if it needs to be recalculated. The snapshot is considered valid if the temperatures are equal to within five signficiant figures. The primary benefit is a speedup of simulations that are isothermal or steady-state in temperature.
update(temperature, thermoData)¶
Update the thermodynamics snapshot to a new temperature (in K) using the information in thermoData. The latter can be any object that provides the necessary getHeatCapacity(), getEnthalpy(), getEntropy(), and getFreeEnergy() methods.

10.2. Methods in rmg.species¶

rmg.species.getThermoData(struct, required_class=<class 'rmg.thermo.ThermoNASAData'>)¶

Get the thermodynamic data associated with structure by looking in the loaded thermodynamic database.

required_class is the class of thermo object you want returning; default is ThermoNASAData

rmg.species.makeNewSpecies(structure, label='', reactive=True)¶

Attempt to make a new species based on a chemical structure, which is a Structure object.

The proposed species is checked against the list of existing species; if the species already exists, this function returns the existing species. If the species does not exist, a Species object is created and returned after being appended to the global species list.
