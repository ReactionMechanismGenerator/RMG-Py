RMG – Reaction Mechanism Generator

RMG is an automatic chemical reaction mechanism generator that constructs kinetic models composed of elementary chemical reaction steps using a general understanding of how molecules react.

The original RMG, written in Java, is described at http://rmg.sourceforge.net/ and developed on github as RMG-Java.

These pages describe the newer Python version of RMG that we call RMG-Py. It is developed on github as RMG-Py.
